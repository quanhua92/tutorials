ComfyUI Workflow — node-graph executor (interactive) 📖 COMFYUI_WORKFLOW.md 📄 comfyui_workflow.py

A ComfyUI workflow is a DAG of typed nodes. Each link [src_id, slot] is a directed edge. ComfyUI topo-sorts the graph and runs the nodes in dependency order. Click Execute to watch nodes light up in topo order; click any node to inspect its inputs/outputs; edit the JSON to rebuild the graph. All values recomputed live in JS — identical to comfyui_workflow.py.
