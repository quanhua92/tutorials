The aHash pipeline — Frame A (vertical split)

Step 1 resize 8×8 → Step 2 grayscale (one luma 0..255 per cell) → Step 3 mean → Step 4 bit = pixel > mean. Green border = bit 1 (above mean).

Hamming distance → cache decision (threshold = 5)

RFC §11: "did this edit change the output?" Diff the perceptual hashes. distance ≤ threshold → HIT (visually equivalent, reuse render); > threshold → MISS (re-render).

A byte hash (SHA-256) would MISS on all three because any 1-bit input diff avalanches ~half the output bits. aHash is locality-sensitive: B ≈ A (HIT), C ≠ A (MISS).
