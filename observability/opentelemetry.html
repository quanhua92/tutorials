② Context Propagation — W3C traceparent across services

traceparent = 00-<traceId>-<spanId>-<flags>. The trace_id stays constant for the whole trace; the span_id changes at each hop. Edit the header below to parse it.
Request flow: Gateway → Auth → Order. Each service extracts the incoming header, creates a child span, injects a new header.
