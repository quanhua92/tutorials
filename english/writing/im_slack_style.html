6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
No greeting → posts bare message ("sent the file") — feels abruptOpen with one of the four openers: Hey / Quick one / Heads up / fyi. Name the stakes in 2–4 words.
Over-formalizes chat — "Dear team, I hope this finds you well…" in a DMRecognize the hybrid register: chat is casual-er than email. Use contractions; drop "Dear"/"Sincerely".
Replies in the main channel instead of the thread (Zalo has no real thread)Click the parent message → Reply in thread. Treat the main channel as a noticeboard.
Overuses @channel / @here ("everyone should see this")Default to @name. @here only for "whoever's online now." @channel only for time-sensitive whole-team news.
Emoji as decoration (dense, like Zalo/Messenger habit)Emoji as tone — one per message max in a work channel. Never in bad-news messages.
Writes essay-length messages (one wall of text)One idea per message. Break long updates into 2–3 short lines or a thread. Scannable beats complete.
No abbreviation register awareness — asap/fyi/brb in formal announcementsKeep initialisms in casual channels/DMs. Write as soon as possible in all-hands or client-facing messages.
Misses the "no hello" rule — sends "Hi" then waits before askingLead with the ask: "Hey, quick one — do you have the Q3 numbers?"
Pro-drop habit → "Is good" / "Will send" (no subject)Supply the subject: "It's good." / "I'll send it." Chat tolerates lowercase, not missing subjects.
