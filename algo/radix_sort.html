① The radix dial — fewer buckets, more passes (work = d·n)

What to do: change the base. More buckets (higher radix) = fewer passes d but more bucket bookkeeping; fewer buckets = more passes. Total work ≈ d·n.

d = floor(logradix(max)) + 1; work ≈ d · n. Higher radix trades bucket-array size for fewer passes.
