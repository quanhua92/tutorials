5Writing task

Write a polite complaint email opener. Start with "I'm afraid there's an issue with…", state the fact, then hedge the ask. Then reveal the model answer.

Model (the full arc: hedge → fact → ask):

I'm afraid there's an issue with the laptop I ordered on Monday. /aɪm əˈfreɪd ðerz ən ˈɪʃuː wɪð/ — the soft opener

It isn't working — it won't turn on, even after a full charge. /ɪt ˈɪznt ˈwɜːrkɪŋ/ — the factual statement, no emotion adjective

Could you look into this and let me know the next steps? /kʊd juː lʊk ˈɪntuː/ — the modal ask, not an imperative

6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
Suppresses the complaint to save face (VN thể diện) → the problem never gets fixedIn English service culture, silence is not politeness. Use the arc: hedge → fact → ask.
Literal imperative translation → "This is broken, fix it." / "Give me refund."Convert to the modal question: "Could you fix it?" / "Could I get a refund?"
Swings blunt OR over-apologetic — no middle gearPractise the arc as three slots: one hedge, one fact, one modal ask. Never bare fact; never apology-buried ask.
Over-apologises for a fault that isn't theirs → "I'm so sorry, so sorry, but… maybe it's a little broken?"Cut the apology (the fault is the seller's). State the fact plainly: "I'm afraid there's an issue with… it isn't working." Let the agent apologise.
Drops the hedge entirely → "There is a problem with this." (sounds like an accusation)Keep the hedge slot filled: "There seems to be a problem…" / "I'm afraid there's an issue…"
/θ/ → /t/ or /d/ on there's → "dere's an issue"Tongue-between-teeth for /ð/ in there's /ðeəz/.
/ʃ/ → /s/ on issue → "iss-yoo" (Vietnamese has no /ʃ/)Drill issue /ˈɪʃuː/ with the blade raised toward the palate.
Drops final /t/ in sort this out → "sort this ou"Release the final /t/ in out. Drill final consonants.
Stress-shift on refund ignored → "I want a re-FUND" (verb stress on a noun)Noun: REFUND /ˈriːfʌnd/ ("I'd like a refund."); verb: re-FUND /rɪˈfʌnd/ ("Could you refund me?").
Uses emotion adjectives → "This food is very bad!" (sounds like a child's tantrum)State the fact, not the feeling: "This isn't warm." / "It hasn't arrived."
