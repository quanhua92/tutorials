5Writing task

Write one sympathy line. Match the weight — pick the right register. Then reveal the model answer.

Prompt: a friend texts "I failed the interview." Reply with sympathy (acknowledge the news).

Model (weight must match the news):

I'm so sorry to hear that. /aɪm soʊ ˈsɑːri tə hɪr ðæt/ — acknowledge (move 1)

That sounds rough. /ðæt saʊndz rʌf/ — react, informal + warm (move 2)

For a death instead, switch register (do NOT mix):

I'm sorry for your loss. My condolences. /aɪm ˈsɑːri fɔːr jɔːr lɑːs/ — death register only

6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
Goes silent — no ready sympathy formula, so you say nothingDeploy the chunk: I'm so sorry to hear that. Silence reads as cold; the acknowledgement is the minimum social obligation.
Uses bare I'm sorry for sympathy → sounds like you caused it (an apology)Extend it: I'm so sorry to hear that. The extension flips sorry from apology → sympathy.
Jumps to advice ("You should apply again!") before acknowledging feelingRun the three moves first (acknowledge → react → name feeling). Advice comes after empathy, never instead of it.
Uses condolences for non-death (lost job, illness) → sounds like the person diedCondolences / deepest sympathies are death-only. For a lost job: I'm so sorry to hear that.
Over-uses chia buồn (funeral-formal) register for everyday bad newsMatch the weight: everyday bad news → That sounds rough / awful (warm, not ritualized).
Says I'm sorry for your loss in casual chat (too heavy for the moment)Sorry for your loss is the death line. For a failed exam: That's such a shame.
Drops finals → devastated → "devasta", condolences → "condolence"Hold every final: -ated /eɪtɪd/, -ces /ɪz/. Drill them tight.
Translates bùn quá literally → "I'm sad for you" (clunky)Use the native empathy structure: You must be devastated / That must be hard. (must be + adjective).
/θ/ in sympathy → /s/ → "sympasy"sympathy /ˈsɪmpəθi/ has a /θ/. Tongue between the teeth.
