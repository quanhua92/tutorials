Topological Sort — interactive TOPOLOGICAL_SORT.md topological_sort.py

Linearise a DAG: DFS pushes nodes on finish (post-order) and reverses the stack; Kahn peels in-degree-0 nodes from a queue. Both are O(V+E). Step through each, then toggle a back-edge to watch cycle detection fire.  |  Tarjan SCC →

① The worked DAG — 6 nodes, "u before v" edges

A diamond at 1,2 → 3 and a chain down to 5. Every edge u→v means "u must come before v." 0 is the only source (in-degree 0).
source (in-deg 0) internal node back-edge (cycle)

② DFS topological sort — push on finish, reverse the stack

What to do: step through the DFS. Gold = currently entering; green = finished & pushed. The result stack (bottom-to-top) reverses into the topological order.

③ Kahn's algorithm — peel in-degree-0 nodes (BFS)

What to do: step through the peel. Each teal node is in the ready queue; green = output. The number inside each node is its current in-degree.

④ Cycle detection — add back-edge 5→2, output < V

What to do: toggle the back-edge 5→2 (closes the cycle 2→3→4→5→2). Kahn's output shrinks below V — the remaining nodes are stuck in the cycle (their in-degree never hits 0).
toggle to add/remove the back-edge 5→2
