Helm Chart — interactive HELM_CHART.md helm_chart.py

A chart is a folder of templates with holes ({{ .Values.replicas }}); values.yaml is the default fillings; helm install runs the mail-merge into finished YAML. Slide the --set image.tag override and toggle the subchart/auth switch below — every rendered line is recomputed live in JS from the identical chart + values as helm_chart.py.
