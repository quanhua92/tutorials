The handler (sketch)

linecode
1_player.addEventListener('timeupdate', () => {
2  const t = _player.currentTime;
3  ext_music.currentTime = t % 30;       // modulo (looped)
4  ext_voiceover.currentTime = Math.min(t, 70); // clamp (not looped)
5});
The player cannot mute ext_music / ext_voiceover — they are independent HTMLMediaElement nodes (AGENTS.md: "bypasses the player's internal muting").
