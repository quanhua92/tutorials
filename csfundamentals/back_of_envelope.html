② Latency Comparison — L1 cache vs disk vs network (log scale)

Six orders of magnitude separate L1 (~1 ns) from cross-region RTT (~200 ms). Bars are log-scaled (each step ≈ 10×). The “if 1 ns = 1 s” column makes the gaps visceral: an SSD random read feels like a day vs DRAM's minute and a half.
CPU/cache ≤1µs  memory/SSD ≤1ms  disk/DC ≤10ms  network >10ms
Non-obvious: intra-DC network (500µs) is faster than a 1 MB SSD sequential read (1 ms) — a distributed in-memory cache can beat a local SSD cache on latency-sensitive paths.

③ Power-of-2 Reference — binary units (storage) vs decimal (network)

Storage & memory use binary multiples (2n); networks & drive vendors use decimal (1 GB = 109 B). That is why a “1 TB” disk shows ~931 GiB.
