Why it works this way — the load-bearing rules

RuleMechanismWhat breaks without it
scale = min(vw/cw, vh/ch)Uniform scale; limiting axis wins (object-fit: contain). Letterbox the rest.Non-uniform scale stretches the 16:9 canvas → warped text/captions.
local = clamp(global − start, 0, dur)Translate shared global playhead into the active slide's GSAP time.Negative or past-end seeks freeze/jump the preview.
one active slideMount exactly one composition; swap on boundary cross (RFC §7).Re-rendering the whole timeline every paint → janky preview.
position:absolute;inset:0;Stretch the host div to the scaled canvas (AGENTS.md).Sub-comp collapses to zero size → blank stage.
real-time, not frame-accurateSeek GSAP per rAF paint (variable dt). Export is the rigorous path (§10/§11).Locking preview to 1/fps makes scrubbing sluggish for no fidelity gain.
