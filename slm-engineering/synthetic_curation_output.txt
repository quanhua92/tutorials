========================================================================
  SECTION B: persona + style system prompt (Cosmopedia = topic+seed+style)
========================================================================
A Cosmopedia prompt has THREE components (verified, see _reference.txt):
  (1) the TOPIC          (cluster label / BISAC category)
  (2) the SEED SAMPLE    (a real web/curated page that grounds the output)
  (3) the GEN. STYLE     (audience + format)

Cosmopedia prompt multiplication: 4 audiences x 3 styles = 12 prompts/topic.
  4 audiences x 3 styles = 12x distinct prompts per topic
  [check] Cosmopedia prompt multiplication is 4x3=12: OK

  -> 3 personas x 2 styles = 6 (topic, persona, style) tuples per topic

words drawn from the persona's adjective pool by slot i = pool[(seed+i) % len(pool)].
