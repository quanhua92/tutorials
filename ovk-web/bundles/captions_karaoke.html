Live karaoke — GSAP direct color tween, driven by the playhead

The CORRECT pattern: two tl.to(sel,{color:...}) tweens per word (color → #ffea00 at start, → dim at end). Drag the slider — it calls tl.time(t). The active word's color tweens; its size/scale/shadow NEVER change.
playhead: t=0.000s active: —
word--emphasis (bottles, planet) is a STATIC green — set once in CSS, never touched by the timeline. There is intentionally no .word--active CSS rule here.
