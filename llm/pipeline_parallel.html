② Bubble (K-1)/(K+M-1) & peak memory M× vs K×

What this proves: GPipe and 1F1B have the SAME bubble fraction, but 1F1B's peak activation memory is bounded at K in-flight micro-batches (vs M for GPipe). For M >> K this is a huge memory win for free. Drag M up and watch the GPipe bar grow while the 1F1B bar stays flat.

Rule of thumb: keep M ≥ 4*K so bubble ≤ ~16%. At K=4, M=16: bubble = 3/19 = 15.79%.
