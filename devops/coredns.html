③ Search domains + ndots — how a short name resolves

What to look at: a pod's /etc/resolv.conf lists search suffixes. With ndots:5, a short name like web-svc is tried relative first — each search domain is appended until one answers. A name in the wrong namespace misses.

④ NodeLocal DNS cache — repeat queries never leave the node

Why it exists: at scale every DNS query is a UDP packet crossing conntrack; under churn this exhausts conntrack and starves CoreDNS (5s stalls). NodeLocal DNSCache runs a cache DaemonSet on every node. Toggle it and watch the path change.
