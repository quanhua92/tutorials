errors_exceptions.ts — Phase 1 bundle.
Every value below is computed by this file; the .md guide pastes
it verbatim. Nothing is hand-computed.

Reminder: JS is throw-based (like Python), NOT return-error (like Go).
Anything can be thrown (always throw an Error); errors propagate up the
call stack until caught; finally ALWAYS runs (even on return/throw).
