③ Drift — out-of-band cloud change surfaces only when refresh runs

  

    
Scenario: someone sets instance_type = t3.large in the AWS console (drift). The config wants
      t3.micro. Flip refresh on: Terraform re-reads the cloud into state before diffing, so the drift appears as an UPDATE.
      Off → the stale state hides it.
