② The min-priority-queue (binary heap) — extract-min in O(log V)

The frontier below is the heap's contents right now, sorted by (dist, node). Each step is one pop (settle/stale) or one push (a successful relaxation). Tie-breaking on equal dist is by node name — matching dijkstra.py Section B exactly.

③ Negative edges break it — and the BFS / Dijkstra / Bellman-Ford verdict

On the main graph (all weights ≥ 0) all three agree on the weight answer; BFS counts edges instead. Add a negative edge C→A = −5 and Dijkstra's greedy commitment silently returns wrong answers — Bellman-Ford still gets it right.
