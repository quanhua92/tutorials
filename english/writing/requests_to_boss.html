★ primaryWriting task — upward request

Write an upward request to your manager: one sentence per move — benefit + two options + one clear ask + a defer. Then reveal the model answer.

1 benefit 2 options (either… or…) 3 clear ask · defer

Model — four moves, one benefit, two options, one clear ask + a defer:

Benefit: It would help the team if we could speed up QA — right now the shared licence is a bottleneck near every deadline.

Options: Here are a couple of options — we could either buy two extra licences, or rotate the three we have across the squads.

Ask: Would it be possible to approve one of these by Friday so we're ready for the next sprint?

Defer: I'd appreciate your guidance on which fits the budget — happy to go with your call.

6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
Never makes the ask — wraps it in hedges ("Nếu được thì… anh xem giúp…") until the request vanishesMake one explicit ask: "Would it be possible to…?". One softener max; the defer-move carries the respect, so the ask can be direct.
Demands with no benefit frame — swings to "Tôi cần X" under pressurePrepend the benefit: "It would help the team if we…". Same ask, now framed as the team's gain.
Brings an open question, not options — "Anh thấy sao?" forces the senior to decideBring two concrete options: "Here are a couple of options — we could either A or B." You analyse; the manager picks.
Stacks three hedges — "If possible, maybe you could perhaps consider…"Calibrate to one hedge: "Would it be possible to…?". Three hedges reads as no confidence, not politeness.
Buries the ask at the end — 3 paragraphs of context, request in the last lineBLUF — put the ask in the first or second sentence. Context can follow; the request must be findable in ≤10 seconds.
Translates L1 ceremonial filler — "Kính gửi sếp… Em xin phép…" word-for-wordUse the upward-request register: "I'd like to propose…" / "Would it be possible to…?". Formal ≠ ornate.
No concrete next step — ends "mong sếp phản hồi" (please reply)Close with one dated next step: "Could we decide by Friday so I can place the order?"
Avoids the request entirely (face) — hints and waitsName it plainly but deferentially: "I'd appreciate your guidance on whether we can approve the licence." Directness wrapped in politeness, not silence.
