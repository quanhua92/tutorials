Killer gotchas (verbatim from monotonic_stack.py Section D):
  1. Store indices, not values: a value stack can't compute distance i - j or rectangle width. Position is NOT recoverable from a value.
  2. Rectangle width = i - stack[-1] - 1 (the bar below the popped one still on the stack), NOT i - top.
  3. The sentinel: append height 0 so the stack fully flushes at the end. Without it, [1,2,3,4,5] never pops in the loop.
  4. Circular = push only on pass 1: loop 2n, push only when i < n. Pushing twice breaks the O(n) invariant.
  5. Ascending vs descending: ascending pops on incoming > top (next greater); descending pops on incoming < top (next smaller).
  6. Strict vs non-strict (< vs <=): distances use strict; subarray-contribution de-dup uses one pass strict + one inclusive.
