① Minrun — the [32,64] dial that balances the merge tree

What to do: drag n. Minrun is chosen so the run count stays near a power of two → a shallow, balanced merge tree → worst case O(n log n).
while (n >= 64) { r |= n&1; n >>= 1; } return n + r; → minrun ∈ [32,64] (CPython). Java uses MIN_MERGE=32 → [16,32].
