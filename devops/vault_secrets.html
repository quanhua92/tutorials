📖 Read the full guide — diagrams, decision tables, common pitfalls, and a cheat sheet. This page is the interactive companion.
Vault is a bank vault for secrets. KV holds static values; the database engine mints short-lived users; transit encrypts without storing; the injector writes secrets to a file the app reads (no SDK). Every ciphertext and TTL below is recomputed live in JS from the identical model as vault_secrets.py.  |  K8s ConfigMap/Secret →

④ Lease lifecycle — renew, revoke, auto-expire

  

    
What to do: drag the clock. The dynamic lease starts at ttl=3600s
      (1h). At t=1800 it's renewed (+3600, bounded by max_ttl=86400 → expiry 5400). At t≥5400 it auto-revokes
      (DROP USER); the revoke button does it early.
