========================================================================
  SECTION D: the operand-order pitfall -- silu(gate)*up vs silu(up)*gate
========================================================================
These look almost identical in code. They give DIFFERENT outputs.

CORRECT  (Llama/Qwen source):  down( silu(gate(x)) * up(x) )
BUGGY    (swap operands):      down( silu(up(x))   * gate(x) )

Same weights, same input, different wiring -> different numbers:

| m | y_correct[0,m,0] | y_buggy[0,m,0] | abs diff |
|---|------------------|----------------|----------|
| 0 |          +0.0120 |        +0.0133 |   0.0013 |
| 1 |          +0.0056 |        +0.0048 |   0.0008 |
| 2 |          +0.0052 |        +0.0077 |   0.0025 |
| 3 |          -0.0042 |        -0.0043 |   0.0000 |

max|y_correct - y_buggy| over the whole output = 0.0035

[check] y_correct == y_buggy?  False  (OK -> they DO differ, confirming operand order matters)

WHY: silu is non-linear, so silu(a)*b != a*silu(b) in general.
The CHECKPOINT was trained with silu on the gate path; swapping
silently corrupts the model with no error message. This is the #1
SwiGLU bug. Fix: read the reference impl -- silu ALWAYS on gate.

========================================================================
  SECTION E: the FFN intermediate ratio (Qwen3-0.5B case)
========================================================================
Qwen3-0.5B config:  E = 896  (hidden_size)
                     FFN = 4864  (intermediate_size)
                     ratio = FFN/E = 5.4286  (NOT 4.0000)

Compare:
  nanoGPT  ratio = 4.0000   (hardcoded 4*E)
  Qwen3-0.5B ratio = 5.4286   (~5.43x)

Why? Llama-class models tune FFN/E empirically (often between 2.6x
and 8x). 4864 is divisible by some common GPU-friendly factors
(4864 = 2^7 * 38). NEVER assume 4x; always
read `intermediate_size` from the model config.

Our tiny model uses E=8, FFN=16 -> ratio 16/8 = 2.0000  (forced 2x so the table is printable; not realistic)
The structure is identical regardless of the exact ratio.
