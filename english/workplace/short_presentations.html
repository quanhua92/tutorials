5Writing task

Signpost a 3-point mini-presentation: First… / Moving on to… / To sum up…. Then reveal the model answer.

Model — 3 points, every phase signposted:

① Let me start with a quick overview of the redesign. (OPEN — roadmap)

② First, our conversion rate climbed eight percent after launch. (SEQUENCE — point 1)

③ Moving on to retention — weekly active users are up for the third month. (SEQUENCE — transition + point 2)

④ Finally, the mobile build ships next Tuesday. (SEQUENCE — last point)

⑤ To sum up: the redesign is working, retention is climbing, and the mobile build is on track. (CLOSE — summary)

6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
Memorises and reads aloud — a flowing script with no improvisationRehearse the structure (the 3 signposts), not the wording. Know First… / Next… / Finally… cold; let the sentences be flexible.
No signposts between points — "Revenue is up. The feature launched. The roadmap is ready." (3 disconnected sentences)Put a signpost before every point: First… / Next… / Moving on to… / Finally…. One signpost per point.
Abrupt ending, no summary — last point spoken, then the talk just stopsAlways close with To sum up… + the one takeaway. The summary is what the audience remembers.
Over-formal scripted tone — "Today I would like to present to you about…"Use plain conversational signposts: Let me start with… / I'll walk you through… / To sum up…. Workplace English is conversational, not ceremonial.
Confuses Finally and To sum up — uses Finally… to start the summaryFinally… = the last point; To sum up… = the summary. Order: First… / Next… / Finally… then To sum up…
Drops the /kst/ cluster in next — "nec" for nextDrill the /kst/ cluster: next /nekst/. 🔗 Final Consonants.
Wrong stress in finally — says fiNALly (last-syllable stress)Stress the first syllable: FI·nal·ly /ˈfaɪnəli/. Vietnamese is syllable-timed, so word stress is easy to miss.
Drops the final /t/ in start — "Let me star with…"Hold the /t/ contact and release audibly: start /stɑːt/. 🔗 Final Consonants.
Treats the close as an apology — trailing off "So… that's all… um, sorry…"Close decisively: To sum up: [one sentence]. That covers everything — any questions? Decisiveness reads as competence.
