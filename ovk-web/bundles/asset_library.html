Dedup — same bytes ⇒ one stored blob

A content-addressable store keys blobs by their hash. Add the same text twice → the second insert is a no-op (orange = deduped). This mirrors asset_library.ts Section C: 3 inputs, 2 byte-identical → 2 stored blobs.
