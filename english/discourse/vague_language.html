5Writing task

Use vague extenders naturally in 3 casual sentences. Then reveal the model — and note which are too casual for formal writing.

Model (3 casual sentences — each uses a vague extender naturally):

1. I've got a lot of stuff to do this weekend.

2. Bring a notebook, a pen, and so on.

3. We talked about the project, the timeline, or whatever.

Too casual for formal writing: #1 (stuff) and #3 (or whatever — sounds dismissive). Formal rewrites: #1 → "I have several tasks to complete…"; #3 → "…the project, the timeline, etc." Sentence #2 (and so on) is borderline but usually acceptable.

6L1 pitfalls & cheat sheet

Vietnamese → English pitfalls

Vietnamese trap (what you do)English fix (what to do instead)
Over-specifies — exhaustive lists ("cleaned my room, washed the dishes, watered the plants…") because class trained you to always be preciseDrop to a vague extender when precision isn't needed: "a bunch of stuff — cleaning and so on." Warmth beats completeness in casual talk.
Never uses stuff — reaches for stiff words ("items", "objects", "activities")Treat stuff as the default casual general noun — Cambridge: "one of the most common nouns in speaking."
Treats stuff as countable → "many stuffs", "a stuff"stuff is uncountable — "a lot of stuff", never "stuffs".
Uses casual vague language in the WRONG register → "and stuff" / "or whatever" in a formal emailSwitch to the formal rung: etc., such as, and so forth. stuff is spoken-only — "not at all common in writing."
Avoids vagueness trying to always be exact → sounds robotic / textbookBe vague on purpose. When the listener knows the category, name the category, not every member.
Freezes when a word won't come ("ờ… cái gì nhỉ…")Plug the gap with thingy / whatchamacallit + a one-clause description, then keep going.
Maps Vietnamese vân vân → etc. everywhere, even in speechetc. is mostly written/formal. In casual speech prefer and so on, and stuff, or or whatever.
Uses Whatever. (flat, one word) as a neutral "anything" → sounds rudeWhatever. (flat tone) signals disrespect, not "anything is fine." For that say whatever you like / either is fine.
