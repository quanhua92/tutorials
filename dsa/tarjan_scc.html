📖 Full guide  ·  📄 Ground-truth source — every panel below recomputes the same model in JS.
Find all Strongly Connected Components in ONE DFS pass. Each node gets a disc (discovery time) and a low (low-link). When disc[v] == low[v], v roots an SCC — pop the stack. Step through, watch the low-link values drop on back-edges.  |  ← Topological sort

① The worked graph — 6 nodes, 3 SCCs (condensation is a chain)

SCC {0,1,2} (a 3-cycle) → SCC {3,4} (a 2-cycle) → SCC {5} (singleton). Each SCC is a maximal set of mutually-reachable nodes; the condensation (shrink each SCC) is always a DAG.
SCC {0,1,2} SCC {3,4} SCC {5} cycle-closing back-edge

② Tarjan's DFS — disc, low-link, and the SCC pop

What to do: step through the single DFS. Each node shows disc/low. Gold = just entered; purple = on the stack; red dashed = a back/cross edge that drops a low-link value. When disc==low, the SCC is popped.

③ The root test — disc[v] == low[v] ⟹ v roots an SCC

A node roots an SCC exactly when its discovery time equals its low-link value. The roots are 0, 3, 5 — they are where each SCC "closes".

④ Cycle detection — SCCs of size > 1 (or self-loops)

Any SCC of size > 1 IS a directed cycle. {0,1,2} and {3,4} are cycles; {5} is a singleton with no self-loop, so it is NOT a cycle.
