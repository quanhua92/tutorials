========================================================================
  SECTION A: parse the prompt JSON -> nodes
========================================================================
ComfyUI prompt format: a dict keyed by node id (string). Each value has
  class_type : the node's operation name (a registered Python class)
  inputs     : dict of input_name -> value OR link [src_id, slot]
A link [src_id, slot] means 'take output slot #slot of node src_id'.

prompt JSON (first node):

  {"class_type":"CLIPTextEncode","inputs":{"text":"beautiful landscape","clip":["4",1]}}

========================================================================
  SECTION C: topological sort (Kahn's algorithm, numeric tie-break)
========================================================================
A node can only execute once ALL its input producers have executed.
Kahn's algorithm: emit zero-in-degree nodes, decrement consumers, repeat.
Tie-break by numeric node id so the order is deterministic.

in-degree per node:
  node 3  (KSampler              ) in-degree = 4
  node 4  (CheckpointLoaderSimple) in-degree = 0
  node 5  (EmptyLatentImage      ) in-degree = 0
  node 6  (CLIPTextEncode        ) in-degree = 1
  node 7  (CLIPTextEncode        ) in-degree = 1
  node 8  (VAEDecode             ) in-degree = 2
  node 9  (SaveImage             ) in-degree = 1

topo-sorted execution order:
  1. node 4 -> CheckpointLoaderSimple
  2. node 5 -> EmptyLatentImage
  3. node 6 -> CLIPTextEncode
  4. node 7 -> CLIPTextEncode
  5. node 3 -> KSampler
  6. node 8 -> VAEDecode
  7. node 9 -> SaveImage

order = ['4', '5', '6', '7', '3', '8', '9']

========================================================================
  SECTION D: execute in topo order (symbolic trace)
========================================================================
Run graph_compute: for each node in topo order, read resolved inputs and
produce typed outputs. Real ComfyUI swaps each node for the actual model
op (load weights, run CLIP, denoise 20 steps, VAE decode, write PNG).

| step | node | class_type             | outputs                              |
|------|------|------------------------|--------------------------------------|
| 1    | 4    | CheckpointLoaderSimple | MODEL(model.safetensors)             |
| 2    | 5    | EmptyLatentImage       | LATENT(512x512 noise)                |
| 3    | 6    | CLIPTextEncode         | COND('beautiful landscape' via CLIP(model.safetensors)) |
| 4    | 7    | CLIPTextEncode         | COND('blurry, bad' via CLIP(model.safetensors)) |
| 5    | 3    | KSampler               | LATENT(denoised seed=42 steps=20)    |
| 6    | 8    | VAEDecode              | IMAGE(512x512x3 pixels)              |
| 7    | 9    | SaveImage              | SAVED(ComfyUI_00001_.png)            |
