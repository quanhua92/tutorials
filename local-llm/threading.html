② Why prefill scales but decode doesn’t (roofline)

Arithmetic intensity = FLOPs / bytes-moved. Decode (batch=1) has intensity ≈ 4 → the memory bus is the ceiling. Prefill (batch=512) has intensity ≈ 1365 → the cores are the ceiling. Threads parallelise compute but share the bus.

③ NUMA awareness — 2-socket server, toggle --numa

A 2-socket server has 2 NUMA nodes, each owning half the DRAM. Without --numa, threads wander across nodes and read weights over the slow interconnect (red). With --numa isolate, each thread stays on the node where its weights live (green).
