📖 Full guide in the .md file  ·  📄 Ground-truth source code in the .py file.
Two ways to explore a graph, both O(V+E). BFS uses a queue and ripples outward in rings (finds the shortest unweighted path). DFS uses a stack and plunges deep (finds cycles, topo order, edge types). Toggle below and step through. All numbers are recomputed live in JS from the identical logic as bfs_dfs.py and gold-checked against it.
