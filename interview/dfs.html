Killer gotchas (verbatim from dfs.py Section D):
  1. MARK BEFORE YOU MOVE — sink the cell before recursing, or neighbours ping-pong forever (stack overflow).
  2. Sink in place ('1'−>'0') instead of a separate visited set.
  3. Check bounds before indexing grid[r][c] or you get IndexError.
  4. Python recursion limit is ~1000 — a snake island can overflow; use sys.setrecursionlimit.
  5. P695 returns the area (1 + sum(neighbours)); P200 returns nothing.
  6. P572 needs both is_same_tree and is_subtree; short-circuit with or.
  7. P572 edge case: subRoot is None is always True — handle it first.
