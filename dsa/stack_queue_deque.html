Each panel steps through a classic algorithm whose correctness DEPENDS on that discipline. What to do: step through. Two stacks — VALUES (numbers) and OPS (operators). A stacked operator with precedence ≥ the incoming one fires first, so 4 * 2 is evaluated before the +. Result pins at 11 (= 3 + 8).

② Queue (FIFO) — BFS frontier on a 6-node graph

What to do: step through. The queue is the BFS frontier: dequeue one node to visit, enqueue its unvisited neighbors. FIFO order is what makes BFS visit nodes in distance order. Visit order pins at [0, 1, 2, 3, 4, 5]. green outline = front, orange outline = back.

③ Deque — sliding-window maximum, k = 3

What to do: step through. A monotonic deque stores indices with strictly decreasing values, so the front is always the window's max. For each new element: evict out-of-window front, pop smaller-or-equal from the back, push the new index, then emit nums[front]. Window slides right each step; result pins at [3, 3, 5, 5, 6, 7].
