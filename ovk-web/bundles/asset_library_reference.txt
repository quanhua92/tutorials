================================================================================
WEB SOURCES (corroborating the cryptographic + storage properties)
================================================================================

7. Wikipedia — SHA-2
   URL: https://en.wikipedia.org/wiki/SHA-2
   Verifies: SHA-256 produces a fixed 256-bit (32-byte / 64-hex-char) digest.
   Verifies: SHA-2 is deterministic — identical input always yields identical
     output.
   Verifies: the avalanche effect — a 1-bit change in input produces a hash
     with no correlation to the original (~50% of output bits change).

8. MDN — SubtleCrypto.digest()
   URL: https://developer.mozilla.org/en-US/docs/Web/API/SubtleCrypto/digest
   Verifies: SHA-256 is one of the standard algorithms exposed by Web Crypto
     (`crypto.subtle.digest('SHA-256', data)`); output is 256 bits / 64 hex
     chars. Confirms the gold value computed in-browser by asset_library.html.

9. Wikipedia — Content-addressable storage
   URL: https://en.wikipedia.org/wiki/Content-addressable_storage
   Verifies: content-addressable storage (CAS) keys blobs by a hash of their
     CONTENT rather than by a location/name.
   Verifies: CAS gives automatic deduplication — identical content yields the
     same key, so the duplicate need not be stored twice. This is exactly the
     mechanism asset_library.ts Section C demonstrates.
