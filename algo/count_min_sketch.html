③ Memory — CMS is O(d·w) regardless of the universe U

What to do: drag eps/delta. CMS memory is set by the error budget, not by how many distinct ids exist. Exact counting needs O(U) — for 64-bit ids that is 18 exabytes; CMS is a flat few KiB.
