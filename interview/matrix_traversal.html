Killer gotchas (verbatim from matrix_traversal.py Section D):
  1. SPIRAL DOUBLE-COUNTING — guard the bottom & left legs with if top≤bottom / if left≤right, or a 1×N strip is read twice.
  2. TRANSPOSE BOUNDS — inner loop must be for j in range(i+1,n); starting at 0 swaps each pair twice and undoes itself.
  3. ROTATE ORDER MATTERS — transpose+reverse rows = 90° CW; reverse+transpose = CCW.
  4. DIAGONAL START CELL — even d starts at row=min(d,m-1); odd d at col=min(d,n-1).
  5. DIAGONAL PARITY is a convention — even d = up-right matches LeetCode P498; check against the example.
  6. P048 is IN PLACE — transpose+reverse is O(1) extra space.
  7. EMPTY / 1×N — short-circuit if not matrix or not matrix[0] or len(matrix[0]) IndexErrors on [].
